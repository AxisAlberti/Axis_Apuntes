# Objetivo del repositorio

<div style="text-align: center;">
  <img src="assets/logo.png" alt="Logo" style="max-width: 240px; height: auto;">
</div>

Este repositorio reune informacion y materiales didacticos para el alumnado de
los modulos **Sistemas Informaticos** y **Montaje y Mantenimiento de Equipos
Informaticos**. El objetivo es ofrecer una base clara y organizada de teoria,
recursos y ejemplos para apoyar el estudio y la practica en el aula.
