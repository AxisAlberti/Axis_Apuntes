# Unidad 00 - Sistemas de numeracion

## Resumen de la unidad

En esta unidad se trabajan los sistemas de numeracion mas usados en informatica (binario, decimal y hexadecimal) y las conversiones entre ellos. Es la base para entender representacion de datos, direcciones de memoria y operaciones a bajo nivel.

## Que aprenderas

- Diferenciar binario, decimal y hexadecimal
- Convertir numeros entre sistemas
- Interpretar valores habituales en hardware y software
