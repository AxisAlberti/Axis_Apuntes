# Unidad 09 - Tarjetas gráficas

## Resumen de la unidad

Esta unidad explica qué es una tarjeta gráfica, cómo funciona y cómo se integra en el PC. Se analizan sus componentes, la memoria de vídeo, las interfaces de salida y los criterios de selección para un montaje correcto.

## Qué aprenderás

- Diferenciar GPU integrada y dedicada
- Identificar los elementos clave de una tarjeta gráfica
- Comprender el flujo básico del renderizado 3D
- Conocer interfaces de conexión (PCIe, HDMI, DisplayPort)
- Aplicar criterios de selección y mantenimiento
